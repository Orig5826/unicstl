# unicstl

## 简介
全称: Universal C standard library

基于C语言实现的通用C库。包含常用数据结构和算法

> 标准：--std=c99

## 数据结构
|数据结构 | 原理 |说明 |
|---|---|---|
| **stack**   | | **栈** |
| stack_init  | 链表 | |
| stack_init2 | 动态数组 | |
| **list**   | | **列表**
| list_init2  | 动态数组 | |
| **queue**   | | **队列**
| queue_init  | 单向链表 | |
| queue_init2 | 数组 | FIFO/空/满 |
| **deque**   | |**双端队列** |
| deque_init  | 双向循环链表 | |
